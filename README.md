## Install

### 1. Install dependencies

```shell
yarn
```

## Start

### Development mode

Create config.json (sample is gii-fn-config.json.sample)

```shell
yarn start:dev /path/to/config.json
```

### Production mode

1. Build

```shell
yarn build
```
2. Start

```shell
yarn start:prod /path/to/config.json
```
